Sunt ergo secundum super comprehensum modum toni octo et modi quattuor quos paracteres, id est, circum aequales appellant per quos omne /[36r] genus modulationis regitur et gubernatur.

Haec secundum Graecos necnon etiam quos a nostros dicta sunt. Verum nos nec octonarium numerum in tonorum varietate excedimus nec aliquid subtraendum ratum esse arbitramur.

Poteramus autem de praefatis quattuor tonis quos circum aequales, [142] id est, medios, eo quod inter duos tonos eorum vis versetur nuncupant nonnulla certis rationibus agitare. Sed brevitatis causa transilientes prudenti lectori inquirendum relinquimus.

His omissis responsoria quae in laude nocturnis horis canuntur persaepe dictos ob tonos tonos [sic] discernamus.

[303] APPENDIX TERTIA

[LEIPZIG MANUSCRIPT, FOLIOS 34v - 35r]

Sunt ergo ni fallor principales divisiones tonorum viginti octo iuxta numerum cordarum quae in tetrachorda quina ponuntur in quibus totius musicae perfectionis summa constitit. Volunt autem quidam supra miminimus tonos tantum modo sive differentias esse duodecim, hoc modo: Tonus inquid est totius constitutions armonicae differentia et quantitas quae in vocis accentu sive tenore consistit. Sunt autem numero duodecim cum interpretationibus suis:

1. Authenticus Protus, id est, antiquus sive autoralis primus.

[304] II. Plagi Protus, id est, obliquus primus.

III. Paracter, id est, circum aequalis.

IV. Autenthicus Deuterus, id est, antiquus emendatus.

V. Plagi Deuterus, id est, obliquus emendatus.

VI. Paracter, id est, circum aequalis.

VII. Authenticus Tritus, id est, antiquus particularis.

VIII. Plagi Tritus, id est, obliquus particularis.

IX. Paracter, id est, circum aequalis.

X. Autenthicus Tetrardi, antiquus quartus.

XI. Plagi Tetrardi, obliquus quartus.

XII. Paracter, id est, circum aequalis

Sunt ergo secundum super [305] comprehensum modum toni octo et modi quattuor quos paracteres, id est, circum aequales, appellant, per quos omne genus modulationis regitur et gubernatur.

Haec secundum Graecos necnon etiam quos a nostros dicta sunt. Verum nos nec octonarium numerum in tonorum varietate excedimus nec aliquid subtraendum ratum esse arbitramur. Poteramus autem de praefatis quattuor tonis quos circum aequales, id est medios, eoquod inter duos tonos eorum vis versetur, nuncupant nonnulla certis rationibus agitare. Sed brevitatis causa transilientes prudenti lectori inquirendum relinquimus.
